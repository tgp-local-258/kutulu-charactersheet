【Kutulu Charactersheet Plusを始める手順】

①下記のリンクからNode.jsの推奨版をインストールしてください。
　https://nodejs.org/ja/

②init.batを実行してください。
　コマンドプロンプトが起動します。いろいろ聞かれますが、すべてEnterを押してください。

③init2.batを実行してください。
　コマンドプロンプトが自動で閉じるまで待ってください。

④run.batを実行してください。
※ここでエラーになった場合、ポートの3000番が既にほかのアプリケーションによって使用されています。
　3000番を使用しているアプリケーションを終了するか、index.jsの「const port = 3000」の部分を書き換えてください。
※ここで開かれるコマンドプロンプトは⑥の閉じる時までは絶対に閉じないでください。

⑤ブラウザを開き下記をブラウザから開いてください。
　localhost:3000
※「:」より後の数字はポート番号です。④でポート番号を変更している場合は、その番号に変えてください。

⑥閉じる時は④で開いたコマンドプロンプトでCtrl + Cを入力して閉じてください。
※再度キャラシを開く場合は④と⑤を再度行ってください。

注）init.bat・init2.bat・run.batはすべてWindows対応となっています。
　　Macをご利用の方はお手数ですが、これらの拡張子を「.command」に変更して実行してください。


■■■■■
プログラミング知識のある人向けですが、Herokuにデプロイすればrun.batを実行しなくてもキャラクターシートを作成することができるようになります。
以下、Herokuデプロイ時のファイルの変更点です。

①package.jsonのnameの次に以下を書き加える。
  "engines": {
    "node": "16.14.1",
    "npm": "8.5.0"
  },

②package.jsonのscriptsの中に以下を書き加える。
    "start": "node index.js",

③index.jsの変数portに代入する値を以下のように書き換える。
process.env.PORT || 3000

※①のnodeとnpmに入れている値はそれぞれのバージョンです。正しい値はコマンドプロンプトで「node -v」「npm -v」と入力してご確認ください。
※ここに書いてあることがよく分からない方はスルーしてください。
